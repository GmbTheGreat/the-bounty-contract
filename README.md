# The Bounty Contract

**The Bounty Contract** is a short low-poly Wild West game built using **Zimension**.  
The game focuses on a hired bounty hunter sent by a sheriff to eliminate a dangerous outlaw known as **Crow Master**.

This repository documents how the game was created ‚Äî including story, cutscenes, animations, sound design, and development workflows ‚Äî in simple and easy language.

---

## Game Overview

- **Genre:** Wild West / Action  
- **Style:** Low-poly  
- **Platform:** Web (Zimension)  
- **Gameplay Length:** Short (5‚Äì12 minutes)  

The player takes on the role of a hired bounty hunter who accepts a contract from a sheriff.  
The first part of the game takes place in an abandoned town called **Dustwood**.

## Parts

### Part 1: Dustwood
- Abandoned town environment  
- Arrival by train
